项目介绍
========

项目背景
-------

本项目模板旨在为Python项目提供一个标准化的开发框架，包含了企业级项目所需的所有基础设施和最佳实践。

设计目标
-------

* 标准化项目结构
* 简化开发流程
* 提高代码质量
* 加速项目启动
* 降低维护成本

技术栈
-----

核心技术
^^^^^^^

* Python 3.8+
* Poetry（依赖管理）
* pytest（测试框架）
* Sphinx（文档工具）

代码质量工具
^^^^^^^^^^

* black（代码格式化）
* flake8（代码检查）
* mypy（类型检查）
* isort（导入排序）

CI/CD工具
^^^^^^^^

* GitHub Actions
* pre-commit hooks
* Docker

项目结构
-------

::

    .
    ├── .github/                    # GitHub配置文件
    ├── .vscode/                    # VSCode配置文件
    ├── docs/                       # 项目文档
    │   ├── en/                    # 英文文档
    │   └── zh/                    # 中文文档
    ├── src/                       # 源代码目录
    │   ├── core/                 # 核心代码
    │   ├── utils/                # 工具函数
    │   └── config/               # 配置文件
    ├── tests/                     # 测试目录
    │   ├── unit/                # 单元测试
    │   ├── integration/         # 集成测试
    │   └── performance/         # 性能测试
    └── ...
