项目概述
-------

这是一个面向企业级的Python项目模板，提供了完整的项目结构、开发流程和最佳实践指南。

主要特性
-------

* 完整的项目结构和最佳实践
* 多语言支持（i18n）
* 完整的测试框架
* CI/CD配置
* 文档自动生成
* 代码质量工具
* Docker支持
* 开发环境自动配置

快速开始
-------

1. 克隆项目模板::

    git clone <repository-url>

2. 初始化开发环境::

    make init

3. 安装依赖::

    poetry install

4. 运行测试::

    make test
